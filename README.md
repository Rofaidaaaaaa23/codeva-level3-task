# codeva-level3-task
# 🧠 Level 3: Advanced Machine Learning & Deep Learning

This repository showcases my projects for Level 3, focusing on Natural Language Processing (NLP) and Time-Series Forecasting using Neural Networks.

## 🎭 Task 2: Sentiment Analysis (SVM)
- **Model:** Support Vector Machine (SVC).
- **Process:** Text preprocessing, TF-IDF Vectorization, and PCA for 2D visualization of decision boundaries.
- **Outcome:** High accuracy in classifying text as positive or negative.

## 📈 Task 3: Stock Price Prediction (LSTM)
- **Model:** Long Short-Term Memory (LSTM) Neural Network.
- **Process:** Data cleaning (Null handling), MinMaxScaler, and Time-Step Sequence generation (60-day window).
- **Outcome:** Precise trend prediction for stock prices, as shown in the final visualization.

## 🛠️ Tech Stack:
- TensorFlow / Keras (Deep Learning).
- Scikit-Learn (SVM & PCA).
- Pandas, NumPy, Matplotlib, Seaborn.
